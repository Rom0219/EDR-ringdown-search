# EDR-ringdown-search

Búsqueda directa de desviaciones QNM (Ringdown) de la Teoría EDR en datos reales de LIGO/Virgo.

Este repositorio implementa una búsqueda directa de modos cuasinormales (QNM) utilizando datos reales de LIGO/Virgo provenientes de GWOSC, con el objetivo de detectar posibles desviaciones respecto a la Relatividad General (RG) predichas por la Teoría del Espacio Dinámico Rotacional (EDR).
